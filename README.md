### Run Locally

To run **Landio** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Harsh Pathak/landio.git
```

Windows:

```bash
git clone https://github.com/Harsh Pathak/landio.git
```
